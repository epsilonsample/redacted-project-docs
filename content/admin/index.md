# Admin Guide

This section covers IT Support and administrative functions within Asset Management.

!!! info "Role Requirement"
    The features in this section require the **IT Support** role.

## Admin Views

| View | Function |
|------|----------|
| [ADMIN: Add/Edit User](user-management.md) | Create and modify user accounts |
| [ADMIN: Permissions](permissions.md) | Manage user and district permissions |

## IT Support Role

When you log in with the IT Support role, your home page is **ADMIN: Add/Edit Users**.

### Available Functions

- Add new users to the system.
- Edit existing user information.
- Manage user permissions by district.
- Manage user permissions by individual user.

### Navigation

Access the Permissions view:

1. Click **Permissions View** button on the home page, or.
2. Select **ADMIN: Permissions** from the left menu.
