# Messages

View in-app notifications and messages.

1. Open the navigation menu (click **â˜°**).
2. Click **Messages**

## Message Types

Messages may include:

- System notifications
- Transfer alerts
- Workflow updates
- Important announcements

## Reading Messages

1. Click on a message to view details.
2. New/unread messages are highlighted.
3. Messages are marked as read after viewing.

## Message Indicators

Look for notification badges or indicators that show:

- Number of unread messages
- New message alerts
