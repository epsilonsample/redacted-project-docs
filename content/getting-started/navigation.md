# Navigation

Learn how to navigate the application interface.

## Main Navigation Menu

Access the navigation menu by clicking the **☰** (hamburger) icon at the top left of any screen.

![Navigation Menu Placeholder](../assets/images/placeholder-table-view.svg)

The menu contains:

- **Views** — Screens available to your role.
- **Features** — Common features like Profile, Messages, etc.

## Navigation Elements

### Top Bar

| Element | Function |
|---------|----------|
| **☰ Menu** | Opens the left navigation panel |
| **Home** | Returns to your role's home view |
| **Quick Actions** | Buttons for common tasks (varies by view) |

### Quick Action Buttons

Most views have quick action buttons at the top:

- **Home** — Return to your home view.
- **Find Bits** — Search for bits.
- **Transactions** — View transactions (if available).
- View-specific actions

### Breadcrumbs

Some views show breadcrumbs to help you track your location:

```
Home > Find Bits > Bit Details
```

Click any breadcrumb to navigate back.

## Mobile Navigation

On mobile devices:

- Tap **☰** to open the navigation drawer.
- Swipe right from the left edge to open the menu.
- Tap outside the menu to close it.
