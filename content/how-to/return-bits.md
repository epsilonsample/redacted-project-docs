# How to Send Bits Back to the Warehouse

Return bits from field inventory back to the central warehouse.

## Prerequisites

- Inventory Manager or Field Sales role.
- Bits are in your inventory.
- Bits are not locked.

## Steps

1. Navigate to **Inventory Management** or **My Bits**.
2. Find the bit you want to return.
3. Click on the bit row to open Bit Details.
4. Click **Return to Warehouse** (or **Send**).
5. Select the warehouse location as the destination.
6. Confirm the transfer details.
7. Click **Submit**

## When to Return Bits

Return bits when:

- Customer no longer needs the bit.
- Bit needs inspection or maintenance.
- Reallocating inventory
- End of project or job

## After Returning

- The transfer appears in your **Initiated eTransfers**.
- Warehouse receives it in their **Received eTransfers**.
- Once accepted, the bit is back in warehouse inventory.

## Returning Multiple Bits

1. Use **Scan Bit Serial No** to queue multiple bits.
2. Review all bits in the queue.
3. Select warehouse as destination.
4. Submit all returns together.
