# How to Send Bits to a Field Salesperson

Transfer bits from inventory to a field sales representative.

## Prerequisites

- Inventory Manager role
- Access to Inventory Management view
- Bits are available (not locked)

## Steps

1. Navigate to **Inventory Management**
2. Find the bit(s) you want to send
3. Click on the bit row to open Bit Details
4. Click **Send to Sales** (or similar action)
5. Select the sales representative from the dropdown
6. Select the destination location if prompted
7. Confirm the transfer
8. Click **Submit**

## Sending Multiple Bits

To send multiple bits at once:

1. Use **Scan Bit Serial No** to add bits to the pending queue
2. Review all bits in the queue
3. Select the recipient
4. Submit all transfers together
