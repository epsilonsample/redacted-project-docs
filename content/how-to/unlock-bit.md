# How to Unlock a Bit

Unlocking a bit makes it available for transfers again.

## Prerequisites

- Access to Inventory Management view
- Inventory Manager role
- The bit is currently locked.

## Steps

1. Navigate to **Inventory Management**.
2. Toggle **See Locked Serials** to ON.
3. Find the locked bit using search or filters.
4. Click on the bit row to open Bit Details.
5. Click **Unlock**
6. The bit is now available for transfers.

## When to Unlock

Unlock bits when:

- Processing is complete
- Issue has been resolved.
- Bit is ready for transfer.
- Lock was applied in error.
