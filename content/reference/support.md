# Support

Get help with the application.

## HelpDesk

For technical issues, contact the **HelpDesk**.

### When to Contact HelpDesk

- Application won't start
- Login issues
- Error messages
- Installation problems
- Account lockouts
- Performance issues

### Information to Provide

When contacting support, have ready:

- Your username
- Application version (found in **About**)
- Description of the issue
- Steps to reproduce the problem
- Any error messages (screenshots helpful).

## Common Issues

### Login Problems

- Verify your network credentials.
- Check Caps Lock is off.
- Try resetting your password
- Contact HelpDesk if account is locked.

### Application Not Loading

- Check your internet connection.
- Try closing and reopening the app.
- Restart your device
- Reinstall if necessary

### Data Not Displaying

- Check your filter settings.
- Click **Clear filter-sort** to reset.
- Verify your profile settings (District, Location).
- Refresh the page

### Sync Issues

- Wait a few moments and refresh.
- Check your internet connection.
- Contact HelpDesk if data remains out of sync.
