# Billing Role

The Billing role is designed for users who process service receipts, manage invoicing, and handle billing operations.

## Quick Reference

| Attribute | Value |
|-----------|-------|
| Home View | [eBilling Workbench](../views/ebilling-workbench.md) |
| View-Only Access | No |
| Primary Function | Service receipt processing and invoicing |

## Available Views

As a Billing user, you have access to:

- **[eBilling Workbench](../views/ebilling-workbench.md)** — Your home view for billing operations.
- **[Find Bits](../views/find-bits.md)** — Advanced search for locating specific bits.
- **[Add/Edit Customer Rep](../views/add-edit-customer-rep.md)** — Manage customer representative records.

## Getting Started

### Logging In

1. Navigate to the login page.
2. Enter your **Username** and **Password**.
3. Click **Log In**

You will be taken directly to the eBilling Workbench view.

### Navigation

Access additional views through the left navigation menu:

1. Click the **☰** (three lines) icon at the top left.
2. Select the view you want to access.

## Common Tasks

### Billing Operations

| Task | Description |
|------|-------------|
| Review ticket lines | Filter and review service receipt ticket lines |
| Edit well information | Update well data on Billed and Reviewed/Approved tickets |
| Add stamped PDF/Photo | Attach documentation to tickets |
| Update National Account Rep | Assign account representatives |
| Print SR PDFs | Generate billing and bit run reports |

### Permissions by Status

| Status | Well Info | Add PDF/Photo | Update Acct Rep |
|--------|:---------:|:-------------:|:---------------:|
| Billed | Edit | Edit | Yes |
| Reviewed/Approved | Edit | Edit | Yes |
| Approved & Locked | View | View | No |
| Invoiced | View | View | No |

## eBilling Filters

Use these filters to find specific ticket lines:

- **Active Tickets** — Current active tickets.
- **Show History** — Invoiced tickets.
- **Approved and Locked** — Locked tickets.
- **Created/Updated Date Range** — Filter by date.
- **Serial Number** — Search by serial.
- **Ready to Review** — Tickets pending review.
