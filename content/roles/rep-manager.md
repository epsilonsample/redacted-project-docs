# Rep Manager Role

The Rep Manager role is designed for users who manage customer representative records.

## Quick Reference

| Attribute | Value |
|-----------|-------|
| Home View | [Add/Edit Customer Rep](../views/add-edit-customer-rep.md) |
| View-Only Access | No |
| Primary Function | Customer representative management |

## Available Views

As a Rep Manager, you have access to:

- **[Add/Edit Customer Rep](../views/add-edit-customer-rep.md)** — Your home view for managing customer reps.

## Getting Started

### Logging In

1. Navigate to the login page.
2. Enter your **Username** and **Password**.
3. Click **Log In**

You will be taken directly to the Add/Edit Customer Rep view.

### Navigation

Access the navigation menu:

1. Click the **☰** (three lines) icon at the top left.
2. Select available options

## Common Tasks

### Managing Customer Representatives

| Task | Description |
|------|-------------|
| Add new rep | Create a new customer representative record |
| Edit existing rep | Update customer representative information |
| Search reps | Find specific customer representatives |
