# Viewer Role

The Viewer role provides read-only access to bit information and historical transaction data.

## Quick Reference

| Attribute | Value |
|-----------|-------|
| Home View | [Find Bits](../views/find-bits.md) (Lite) |
| View-Only Access | **Yes** |
| Primary Function | View bit information and history |

## Available Views

As a Viewer, you have access to:

- **[Find Bits](../views/find-bits.md)** (Lite) — Your home view for searching bits.
- **[Nav History Transactions](../views/nav-history.md)** — View historical transaction data.

!!! info "View-Only Access"
    The Viewer role has read-only access. You cannot modify data, create tickets, or perform transfers.

## Getting Started

### Logging In

1. Navigate to the login page.
2. Enter your **Username** and **Password**.
3. Click **Log In**

You will be taken directly to the Find Bits (Lite) view.

### Navigation

Access additional views through the left navigation menu:

1. Click the **☰** (three lines) icon at the top left.
2. Select the view you want to access.

## Common Tasks

### Searching for Bits

1. Go to **Find Bits** view
2. Set your search filters:
   - Serial or Item Number
   - Bit Status
   - Company
   - Location filters
3. Click **Search**
4. Click any row to view bit details.

### Viewing Transaction History

1. Go to **Nav History Transactions**
2. Set your filters:
   - District
   - Location
   - Serial number
   - Date range
3. Click **Apply Filters**

## Limitations

As a Viewer, you **cannot**:

- Create or edit tickets.
- Transfer bits
- Modify bit status
- Update records
