# Views Reference

This section documents each view available in Asset Management. Views are the main screens where you perform tasks within the application.

## View Availability by Role

Not all roles have access to all views. The table below shows which views are available to each role.

| View | Description | Available To |
|------|-------------|--------------|
| [Inventory Management](inventory-management.md) | Manage bit inventory and stock | Inventory Manager |
| [My Bits](my-bits.md) | View and manage your assigned bits | Field Sales |
| [Find Bits](find-bits.md) | Advanced search for locating bits | Most roles |
| [Transactions](transactions.md) | View bit orders and inventory transactions | Inventory Manager |
| [Nav History](nav-history.md) | Historical transaction data | Inv. Mgr, Viewer, Finance |
| [eTransfer Tickets](etransfer-tickets.md) | Electronic transfer management | Inv. Mgr, Field Sales |
| [eBilling Workbench](ebilling-workbench.md) | Service receipts and billing | Field Sales, Billing, Acct Rep, Finance |
| [Add/Edit Customer Rep](add-edit-customer-rep.md) | Manage customer representatives | Billing, Rep Manager |

## Common View Features

Most views share these common features:

### Navigation Elements

| Element | Function |
|---------|----------|
| **☰ Menu** | Opens the left navigation panel |
| **Home button** | Returns to your role's home view |
| **Quick action buttons** | Jump to related views |

### Table Features

All data tables support:

- **Global search** — Search across all columns.
- **Column sorting** — Click headers to sort.
- **Column filtering** — Click funnel icons to filter.
- **Clear filter-sort** — Reset table display options.

For detailed information on searching and filtering, see [Searching and Filtering Data](../features/search-filter.md).

### Row Selection

Click any row in a data table to:

- View details for that item.
- Access the action menu
- Navigate to related screens.

## Platform Differences

Some features work differently on mobile:

| Feature | Desktop/iPad | Mobile |
|---------|--------------|--------|
| Column sorting | Click column headers | Use Sort Order button |
| Column filtering | Click funnel icons | Limited to main filters |
| Clear filters | Clear filter-sort button | Clear filter button |
