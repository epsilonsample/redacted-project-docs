# Nav History Transactions View

The Nav History Transactions view displays historical transaction data for bits.

!!! info "Role Availability"
    This view is available to: Inventory Manager, Viewer, Finance

## Overview

![Nav History Transactions Placeholder](../assets/images/placeholder-table-view.svg)

Use this view to research the history of bit movements and transactions over time.

## Filters

### Available Filters

| Filter | Description |
|--------|-------------|
| **District** | Filter by geographic district |
| **Location** | Filter by specific location |
| **Serial** | Search by serial number |
| **Look Back Days** | Number of days to search (default: 7) |
| **Start Date/Time** | Beginning of date range |
| **End Date/Time** | End of date range |

!!! note "Default Look Back"
    The Look Back Days filter defaults to **7 days**. Adjust this to see more or less history.

## Using the View

### Step 1: Set Filters

1. Select your **District** (optional).
2. Select your **Location** (optional).
3. Enter a **Serial** number (optional).
4. Set **Look Back Days** or specific date range.

### Step 2: Apply Filters

Click **Apply Filters** to execute the search.

### Step 3: Review Results

Results display in the table at the bottom of the page.

## Table Features

### Search

Use the search box to find specific records in the results.

### Sorting and Filtering

- Click column headers to sort.
- Click funnel icons to filter columns.

### Clearing

Click **Clear filter-sort** to reset table sorting and filtering.

!!! note
    This clears the table display options, not the main filters above.

## Common Uses

- Track where a specific bit has been.
- Research transaction history
- Audit inventory movements
- Investigate discrepancies
