# Transactions View

The Transactions view displays bit orders and inventory transactions.

!!! tip "Quick Actions"
    :material-package-variant: [Process a Bit Order](../how-to/process-bit-order.md) · :material-cart: [Submit a Bit Order](../how-to/submit-bit-order.md) · :material-magnify: [Where Is My Bit?](../how-to/find-my-bit.md)

!!! info "Role Availability"
    This view is available to: Inventory Manager

## Overview

![Transactions View Placeholder](../assets/images/placeholder-table-view.svg)

The Transactions view is divided into two sections:

- **Bit Orders** — Incoming requests for bits.
- **Inventory Transactions** — Record of inventory movements.

## Bit Orders Section

### Viewing Bit Orders

By default, the view shows **Pending** bit orders.

Toggle the switch to view **In Progress/Filled** orders.

### Filtering by Date

Use the date filters to narrow down bit orders by time period.

## Inventory Transactions Section

### Viewing Transactions

Inventory transactions show the history of bit movements.

### Filtering by Date

Use the date filters to view transactions within a specific time period.

## Table Features

### Global Search

Use the search box above either table to search across all columns.

### Sorting and Filtering

1. Click column headers to sort.
2. Click funnel icons to filter by specific values.

### Clearing Filters

Click **Clear filter-sort** to reset table filters and sorting.

### Exporting Data

Click **Export to CSV** to download the filtered data as a spreadsheet.
